# Fin Trac

> Личный трекер доходов и расходов. Сделан онли для себя — не продукт
> без монетизации и без публичного доступа.

Форма добавления операций, список по дням, свои категории и дашборд с
интерактивными графиками. Работает как обычный сайт, но добавляется на
домашний экран телефона и открывается как отдельное приложение.

**React + Supabase + Vercel + GitHub**
*всё на фри тарифах

Открывается как обычный сайт с телефона; можно добавить на домашний экран,
и он будет открываться как отдельное приложение т.е без адресной строки.
